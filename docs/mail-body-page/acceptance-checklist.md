# 邮件正文页验收清单

## 1. 信息展示

- [ ] 邮件主题清晰可见
- [ ] 发件人姓名清晰可见
- [ ] 发件人邮箱清晰可见
- [ ] 发送时间清晰可见
- [ ] 收件关系清晰可见
- [ ] 外部发件人有明确标识
- [ ] 系统邮件有明确标识
- [ ] 重要邮件有明确标识

## 2. 操作效率

- [ ] 回复入口可见
- [ ] 回复全部入口可见
- [ ] 转发入口可见
- [ ] 归档入口可用
- [ ] 删除入口可用
- [ ] 更多操作可用
- [ ] 操作后有反馈
- [ ] 删除或归档后支持连续处理
- [ ] 删除或归档后可撤销

## 3. 附件体验

- [ ] 有附件时展示附件区
- [ ] 无附件时不展示附件区
- [ ] 附件文件名可见
- [ ] 附件类型可见
- [ ] 附件大小可见
- [ ] 附件下载入口可用
- [ ] 附件预览入口可用或有不可用说明
- [ ] 附件风险状态可见
- [ ] 附件扫描中状态可见
- [ ] 多附件支持查看全部

## 4. 安全提示

- [ ] 外部邮件有提示
- [ ] 外部链接有提示
- [ ] 风险附件有提示
- [ ] 高风险邮件可阻断危险操作
- [ ] 安全提示分级明确
- [ ] 安全提示不滥用强警告

## 5. 正文阅读

- [ ] 正文阅读宽度合理
- [ ] 长正文可正常滚动
- [ ] 图片可适配容器
- [ ] 表格不会破坏布局
- [ ] 历史引用默认折叠
- [ ] 签名和免责声明视觉弱化

## 6. 快捷回复

- [ ] 快捷回复入口可见
- [ ] 点击后可进入编辑
- [ ] 发送中禁用重复提交
- [ ] 系统邮件可隐藏快捷回复
- [ ] 回复全部涉及外部人员时有确认提示

## 7. 异常状态

- [ ] 未选中邮件状态
- [ ] 加载中状态
- [ ] 正文加载失败状态
- [ ] 附件加载失败状态
- [ ] 邮件已删除状态
- [ ] 安全拦截状态
- [ ] 外部图片拦截状态
- [ ] 空正文状态

## 8. 工程质量

- [ ] 组件拆分清晰
- [ ] 类型定义清晰
- [ ] Mock 数据不写死在深层组件中
- [ ] 复用已有组件
- [ ] 没有明显重复代码
- [ ] 没有明显 console error
- [ ] TypeScript 检查通过
- [ ] Lint 通过
- [ ] Build 通过
